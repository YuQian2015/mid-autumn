### 启动

#### 下载项目

```powershell
$ git clone https://github.com/YuQian2015/mid-autumn.git
```

#### 启动dev，开发

```powershell
$ cd mid-autumn

$ npm install

$ npm start

```

#### build

```powershell
$ npm run build
```
