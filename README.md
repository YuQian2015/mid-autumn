### 使用

#### 下载项目

```powershell
$ git clone https://github.com/YuQian2015/mid-autumn.git
```

#### 启动开发

```powershell
$ cd mid-autumn

$ npm install

$ npm start

```

#### 编译打包

```powershell
$ npm run build
```

- 返回博饼结果
- 提前设置结果
